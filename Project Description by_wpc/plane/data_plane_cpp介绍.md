## 代码含义说明

生成 $n$ 个城市以及 $m$ 个航班 。

$f$ 数组输出 $1\sim n$ 个城市编号，并且打乱 。

**我对原来的代码生成随机数据的方式进行了修改**，使得每个城市都能至少作为一次出发站和终点站，且航班的结构层次被修改成了环，前 $n$ 次循环构成了一个单向的环, 第二次 $n$ 个循环反向加环，最后 $m-2*n$ 个航班是在环内加边。

以下生成随机数据的方式比较简单，故不做说明。
